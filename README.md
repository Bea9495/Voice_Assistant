Pequeño proyecto para tener un asistente por voz.

El objetivo era poder conectar el micrófono y a través de el, poder interactuar con el asistente. Para el uso del calendario, es necesario crear y activar la API de google Calendar, aunque realmente no es necesario, ya que lo interesante de esto era el poder, como comentaba al principio, interactuar con el ordeadnador. Pudiendo pasar texto a voz y voz a texto.
